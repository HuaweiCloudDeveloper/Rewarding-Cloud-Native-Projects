# 仓库简介<a name="ZH-CN_TOPIC_0000001432277785"></a>

对象存储服务（Object Storage Service,OBS）是一个基于对象的存储服务，为客户提供海量、安全、高可靠、低成本的数据存储能力。华为云团队提供多种工具接入OBS代码示例，并为应用提供ServiceStage一站式解决方案，帮助开发者快速对接华为云服务。
